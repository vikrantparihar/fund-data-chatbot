# Fund Data Chatbot

This project is an AI-powered chatbot that allows users to analyze fund holdings and trades data.

## Features
- View fund holdings data
- View trade data
- Ask questions about the dataset
- Interactive dashboard using Streamlit

## Tech Stack
- Python
- Pandas
- Streamlit

## Project Structure

fund-data-chatbot/
│
├── app.py
├── holdings.csv
├── trades.csv
├── Fund_Data_Chatbot.ipynb
└── requirements.txt

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
